# Ionyx v0.1.1 Has Arrived! 🚀

We are thrilled to announce **Ionyx v0.1.1**, our most significant update to date. This release introduces revolutionary features that redefine how native desktop applications are built.

## 🧬 Native Fusion: Context Unification

Inspired by the workflow of NW.js, **Native Fusion** creates a shared memory space between Rust and JavaScript. No more manual IPC boilerplate. With `window.fusion`, your backend data is right there in the frontend, reactive and synchronized.

## 🎮 Next-Gen Graphics with WebGPU

Ionyx is now fully **WebGPU native**. Every new project template comes pre-configured with WebGPU support, enabling lower overhead and access to modern GPU features for your desktop apps.

## ⚕️ Ionyx Doctor: Zero-Stress Setup

Setting up a desktop development environment can be tricky. **Ionyx Doctor** analyzes your system, verifies your Toolchain (Rust, Node, Bundlers), and gives you actionable advice to get your environment "Green" in seconds.

## 🖼️ Icon Engine & Native Bundling

Distributing your app is easier than ever. One icon source is now automatically converted for all platforms, and `ionyx bundle` creates professional `.exe`, `.msi`, `.dmg`, and AppImage installers with zero configuration.

## 📦 Now on NPM

Ionyx CLI is now officially available as **`@ionyx-apps/cli`**.

```bash
npm install -g @ionyx-apps/cli
ionyx init my-app
```
