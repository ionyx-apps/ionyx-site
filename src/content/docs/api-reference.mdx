# API Reference

Complete API documentation for the Ionyx framework.

## Core API

### ionyx.create(options)

Create a new Ionyx project programmatically.

```javascript
const app = ionyx.create({
  template: 'react',
  name: 'my-app',
  directory: './my-app'
});
```

**Parameters:**
- `template` (string): Frontend template ('react', 'vue', 'svelte', 'vanilla')
- `name` (string): Project name
- `directory` (string): Output directory (optional, defaults to `./{name}`)

**Returns:** Promise<Project>

### ionyx.build(options)

Build your application for production.

```javascript
await ionyx.build({
  target: 'production',
  optimize: true,
  platform: 'all'
});
```

**Parameters:**
- `target` (string): Build target ('development', 'production')
- `optimize` (boolean): Enable optimizations
- `platform` (string): Target platform ('windows', 'macos', 'linux', 'all')

**Returns:** Promise<BuildResult>

### ionyx.dev(options)

Start the development server.

```javascript
ionyx.dev({
  port: 3000,
  host: 'localhost',
  open: true
});
```

**Parameters:**
- `port` (number): Server port (default: 3000)
- `host` (string): Server host (default: 'localhost')
- `open` (boolean): Open browser automatically (default: false)

**Returns:** DevServer

## Window API

### window.setTitle(title)

Set the window title.

```javascript
window.setTitle('My Application');
```

**Parameters:**
- `title` (string): Window title

### window.setSize(width, height)

Set window dimensions.

```javascript
window.setSize(1200, 800);
```

**Parameters:**
- `width` (number): Window width in pixels
- `height` (number): Window height in pixels

### window.center()

Center the window on the screen.

```javascript
window.center();
```

### window.minimize()

Minimize the window.

```javascript
window.minimize();
```

### window.maximize()

Maximize the window.

```javascript
window.maximize();
```

### window.close()

Close the window.

```javascript
window.close();
```

## File System API

### fs.readFile(path)

Read a file from the file system.

```javascript
const content = await fs.readFile('./config.json');
```

**Parameters:**
- `path` (string): File path

**Returns:** Promise<string>

### fs.writeFile(path, content)

Write content to a file.

```javascript
await fs.writeFile('./data.json', JSON.stringify(data));
```

**Parameters:**
- `path` (string): File path
- `content` (string): File content

**Returns:** Promise<void>

### fs.exists(path)

Check if a file or directory exists.

```javascript
const exists = await fs.exists('./config.json');
```

**Parameters:**
- `path` (string): Path to check

**Returns:** Promise<boolean>

## System API

### sys.showNotification(title, body)

Show a system notification.

```javascript
sys.showNotification('Update Available', 'A new version is ready to download');
```

**Parameters:**
- `title` (string): Notification title
- `body` (string): Notification body

### sys.openUrl(url)

Open a URL in the default browser.

```javascript
sys.openUrl('https://github.com/ionyx-apps/ionyx');
```

**Parameters:**
- `url` (string): URL to open

### sys.showMessageBox(options)

Show a message box dialog.

```javascript
const result = await sys.showMessageBox({
  type: 'question',
  title: 'Confirm',
  message: 'Are you sure you want to exit?',
  buttons: ['Yes', 'No']
});
```

**Parameters:**
- `type` (string): Dialog type ('info', 'warning', 'error', 'question')
- `title` (string): Dialog title
- `message` (string): Dialog message
- `buttons` (string[]): Available buttons

**Returns:** Promise<string> - The button that was clicked

## Events

### onWindowClose

Called when the window is about to close.

```javascript
onWindowClose(() => {
  console.log('Window is closing');
  return false; // Prevent closing
});
```

### onWindowResize

Called when the window is resized.

```javascript
onWindowResize((width, height) => {
  console.log(`Window resized to ${width}x${height}`);
});
```

### onFileDrop

Called when files are dropped on the window.

```javascript
onFileDrop((files) => {
  console.log('Files dropped:', files);
});
```

## Configuration API

### config.get(key)

Get a configuration value.

```javascript
const theme = config.get('theme');
```

**Parameters:**
- `key` (string): Configuration key

**Returns:** any

### config.set(key, value)

Set a configuration value.

```javascript
config.set('theme', 'dark');
```

**Parameters:**
- `key` (string): Configuration key
- `value` (any): Configuration value

### config.save()

Save the current configuration.

```javascript
await config.save();
```
