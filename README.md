## ✨ Overview

Bu depo, **Ionyx Framework**'ün resmi web sitesinin ve dokümantasyon merkezinin kaynak kodlarını içerir. Site, modern bir geliştirici deneyimi sunmak için en son web teknolojileriyle (Astro 5 + MDX) inşa edilmiştir.

### 🌟 Özellikler

- **MDX Tabanlı Dokümantasyon**: Kod blokları, interaktif bileşenler ve zengin içerikli rehberler.
- **Çoklu Dil Desteği**: İngilizce (EN) ve Türkçe (TR) dillerinde tam kapsamlı içerik.
- **Performans Canavarı**: Astro'nun "Zero JS" yaklaşımıyla ışık hızında yükleme.
- **Karanlık Mod**: Geliştiriciler için optimize edilmiş, göz yormayan dark-first tasarım.
- **Responsive**: Mobil, tablet ve masaüstü cihazlarda kusursuz görünüm.

## 🚀 Hızlı Başlangıç

### 1. Kurulum

```bash
# Repo'yu klonlayın
git clone https://github.com/ionyx-apps/ionyx-site.git
cd ionyx-site

# Bağımlılıkları yükleyin
npm install
```

### 2. Geliştirme

```bash
# Yerel sunucuyu başlatın (http://localhost:4321)
npm run dev
```

### 3. Build & Production

```bash
# Üretim için derleyin
npm run build

# Build sonucunu önizleyin
npm run preview
```

## 📁 Proje Yapısı

- **`src/content/docs/`**: MDX tabanlı ana dokümantasyon dosyaları.
- **`src/content/blog/`**: Framework güncellemeleri ve blog yazıları.
- **`src/pages/`**: Sayfa yapıları ve dinamik route yönetimi.
- **`src/components/`**: Tekrar kullanılabilir Astro ve UI bileşenleri.
- **`src/layouts/`**: Sayfa düzenleri ve SEO optimizasyonu.
